FACEBOOK AUTO POSTER v5 - GITHUB UPLOAD GUIDE

IMPORTANT:
1. Extract this ZIP.
2. Open the extracted folder.
3. Upload ALL contents to the ROOT of your GitHub repository.
4. The hidden .github folder is required. It contains:
   .github/workflows/auto-post.yml
5. On Windows, enable: File Explorer > View > Show > Hidden items.
6. Do not upload the outer folder as a nested folder.

Correct GitHub root:
.github/
app/
tests/
auto_poster.py
requirements.txt
keywords.csv
posted_log.json
README.md

Required GitHub Actions secrets:
FB_PAGE_ID
FB_ACCESS_TOKEN
FIXED_LINK
