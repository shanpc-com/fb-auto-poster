FACEBOOK AUTO POSTER v5.1 - GITHUB UPLOAD GUIDE

1. Extract this ZIP on your computer.
2. Open the extracted folder.
3. In Windows File Explorer enable: View > Show > Hidden items.
4. Upload every item INSIDE the folder to the repository root.
5. Confirm this exact path exists on GitHub:
   .github/workflows/auto-post.yml

IMPORTANT:
- GitHub Actions only reads .github/workflows/auto-post.yml.
- AUTO-POST-WORKFLOW.yml in the root is a visible emergency copy only.
- Do not keep __pycache__ folders.

If .github is skipped by browser upload:
A. GitHub repository > Add file > Create new file
B. In the filename box type: .github/workflows/auto-post.yml
C. Open AUTO-POST-WORKFLOW.yml from this package, copy all text, paste it there.
D. Commit changes.

Required repository secrets:
FB_PAGE_ID
FB_ACCESS_TOKEN
FIXED_LINK
